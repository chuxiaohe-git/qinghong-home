## 卡片透明度统一与可调节
- **问题**：夜间模式卡片太透明（--bg-glass: rgba(255,255,255,0.05)），白天模式较实（--bg-glass: rgba(255,255,255,0.6)）
- **解决方案**：
  - 新增 CSS 变量 `--card-glass-alpha`（基础透明度，日夜模式统一为 0.6）
  - 新增 CSS 变量 `--card-bg-alpha`（用户可调节倍率，默认 1）
  - 卡片背景改用 `rgba(255,255,255, calc(var(--card-glass-alpha) * var(--card-bg-alpha)))`
- **个性化设置新增滑块**：0.3 ~ 1.0 范围，步进 0.05，保存到 `layout_config.cardBgAlpha`
- **涉及文件**：`src/assets/main.css`、`src/components/center/CardGrid.vue`、`src/components/settings/tabs/PersonalizeTab.vue`
