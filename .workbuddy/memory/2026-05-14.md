# 2026-05-14

## 深色模式便签颜色柔化
- 新增 `NOTE_STYLES_DARK` 数组（深色模式专有，亮度/饱和度降低）
- 修改 `getNoteStyle` 逻辑：浅色模式用原色，深色模式通过 `LIGHT_TO_DARK` Map 映射
- `randomColor` 始终返回浅色值，存储不受主题影响

## Bug 修复：主题切换便签颜色不刷新
- 原 `isDarkMode()` 是普通函数，class 变化不触发 Vue 重渲染
- 改为 `isDark` ref + MutationObserver 监听 `<html>` class 变化
- `getNoteStyle` 引用 `isDark.value`，主题切换时自动触发重渲染

## 性能优化
- 新增 `LIGHT_TO_DARK` Map 代替 `findIndex` 做颜色查找（O(1)）
- SVG filter `numOctaves` 尝试降为 2 被用户否决（恢复 3），感知差异明显
