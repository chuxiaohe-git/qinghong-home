
## 摸鱼日历 UI 重构：去掉玻璃感 → 实体厚重风格

### 需求
用户反馈日历太"轻透"，希望做成老卷轴那种实实在在的质感，不要半透明。

### 改动：CalendarView.vue CSS 重构
1. **全实体背景**：去掉所有 `rgba(255,255,255,0.02~0.08)` 透明层，改为实心背景
2. **day-header 独立实体区块**：
   - 2px 粗底边分割线
   - 独立实心底色 `rgba(255,255,255,0.025)`
   - 星期标签：8px 小方块标签（背景+边框）
   - 日期数字：22px 粗描边 `text-shadow: 0 2px 4px rgba(0,0,0,0.4)`
   - 月份：胶囊形标签
3. **今天列**：实心淡紫背景 + 头部全紫化 + 数字发光
4. **周末列**：实心暖红底 + 头部红调
5. **列分隔**：1px 实线边框，不再用 gap+背景色
6. **底部导航**：实心边框按钮，不再透明
7. **边框加粗**：容器 `border:1px solid`，阴影增强

### 2026-05-16 00:50 — v8 卷轴书页风格最终定稿

经过 v6（彩色侧边条，太荧光）、v7（七彩列，太散乱）的试错，最终确定 **v8 低饱和统一色调**。

**最终改动（CalendarView.vue CSS 重写，不动便签）：**
1. **卷轴书页质感**：容器加 `inset` 顶部白线 + 厚重多层阴影，像堆叠的古书
2. **顶部/底部渐变装饰线**：铜色/暗金色渐变，像古书装帧线
3. **列分隔像书页折痕**：`border-left: 1px solid rgba(0,0,0,0.15)` + `box-shadow: -1px 0 0 rgba(255,255,255,0.02) inset`
4. **今天列对称高亮**：四边都有淡紫边框 + 顶部渐变装饰线（不再单侧失衡）
5. **day-header 质感页眉**：
   - 细线框铜色标签（星期/月份）
   - 底部渐变装饰线（像页眉分隔）
   - 日期数字 26px/900 + text-shadow 粗描边
6. **统一低饱和色调**：所有框架元素用暗铜色/暗金色系，不再七彩荧光
7. **空状态**：虚线边框，hover 铜色高亮
8. **月份显示保持中文**：`{{ d.month }}月`
9. **便签完全不动**：保持原有设计（噪点纹理、卷角、阴影等）
