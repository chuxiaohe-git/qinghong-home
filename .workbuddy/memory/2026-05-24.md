# 2026-05-24 工作日志

## URL 协议自动补全修复

**问题**：保存 `www.github.com`（缺 `https://` 前缀）后，点击打开被当作相对路径拼接到当前域名上。

**根因**：前端保存 → 后端存储 → 前端打开整个链路中，没有任何环节对 URL 做协议检测或自动补全。

**修复**（三处改动）：
1. **前端 `CardGrid.vue` `saveBookmark()`**：保存前检测 URL 是否以 `http://` 或 `https://` 开头，若否则自动补全 `https://`
2. **后端 `routes/bookmarks.py` `create_bookmark()`**：创建时同样做协议补全（兜底）
3. **后端 `routes/bookmarks.py` `update_bookmark()`**：更新 url 字段时也做协议补全（兜底）
