## 项目定位

从静态个人品牌落地页升级为功能性的**个人导航面板**，集成网页收藏管理 + 轻鸿待办网页版。参考 Sun-Panel 的设计理念，三栏布局，右上角设置弹窗。

## 技术栈

Python + Flask + Vue 3 + SQLite + Vite（前后端分离）

## 架构概要

### 布局（三栏）
- **左侧**：分组菜单栏（导航分组列表）
- **中间**：网页收藏卡片网格（用户可自定义设计卡片样式）
- **右侧**：轻鸿待办网页版面板（默认展开，可收起）

### 设置（右上角弹窗）
九个 Tab：
1. **我的信息** — 昵称、密码、语言、主题
2. **个性化设置** — LOGO/壁纸/卡片风格/时钟/搜索栏/页脚
3. **分组管理** — 分组增删改排序
4. **图库** — 上传/筛选/网格展示壁纸和图标
5. **导入导出** — JSON 导入导出配置、浏览器书签转换
6. **AI 设置** — AI API 接口配置（用途待定）
7. **账号管理** — 超级管理员手动添加/管理用户，无注册渠道，数据隔离
8. **全局设置** — 站点标题/登录页背景/自定义JS CSS等
9. **备份恢复** — SQLite 备份与恢复

### 用户系统
- 多用户数据隔离
- 超级管理员手动添加用户
- 无公开注册渠道
